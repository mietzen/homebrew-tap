# Mietzen Tap

## How do I install these formulae?

`brew install mietzen/tap/<formula>`

Or `brew tap mietzen/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mietzen/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
